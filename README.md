# dotfiles

## Install Zsh Autosuggestions

1. Clone the zsh-autosuggestions repository somewhere on your machine. This guide will assume `~/.zsh/zsh-autosuggestions`.

    ```sh
    git clone https://github.com/zsh-users/zsh-autosuggestions ~/.zsh/zsh-autosuggestions
    ```

2. Clone the zsh-syntax-highlighting repository somewhere on your machine. This guide will assume `~/.zsh/zsh-syntax-highlighting`.

    ```sh
    git clone git@github.com:zsh-users/zsh-syntax-highlighting.git ~/.zsh/zsh-syntax-highlighting
    ```
